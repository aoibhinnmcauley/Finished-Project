virtual void BaseApplication::randomShape ( )
protectedvirtual

This method gets a random shape from an option of 6 and then sets a bool to true to say that shape has been used, This is an adaptation of the 7 bag algorithm used in tetris the shape cannot be used again until all shapes bools have been set to true (This creates semi-randomly generated shapes as completely random would make the game unplayable)
